# workbox-install

Windows installer for the Workbox service host. PowerShell script (`install.ps1`)
that provisions a fresh Windows machine: Node.js, NSSM, Workbox itself,
Tailscale join, and a `VIVAWorkbox` service registration.

For architecture, setup, operations, and troubleshooting documentation for
the broader bridge system, see
**[VIVA-Creative/vivabridge](https://github.com/VIVA-Creative/vivabridge)**.

For the Workbox runtime, see
**[VIVA-Creative/workbox](https://github.com/VIVA-Creative/workbox)**.
